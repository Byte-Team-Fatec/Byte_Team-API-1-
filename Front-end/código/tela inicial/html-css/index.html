Seja bem-vindo!

Projeto do API da equipe BYTE desenvolvido com intuito de ajudar eleitores de São José dos Campos na escolha do candidato a vereador ideal. Entendemos como é díficil tomar decisões que irão afetar toda a população, por isso fizemos esse software com informações diretamente do site da câmara para auxiliar a todos tomarem decisões mais embasadas e acertivas!
